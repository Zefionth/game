# Vampire Survivors Clone

## Описание
Это клон игры Vampire Survivors, написанный на Python с использованием библиотеки Pygame.

## Установка
1. Установите зависимости:
    ```bash
    pip install -r requirements.txt

2. Запустите игру:

    ```bash
    python src/main.py
